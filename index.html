DevTerm

DevTerm adalah aplikasi ensiklopedia teknologi informasi. Aplikasi ini dibuat menggunakan Jetpack Compose dan untuk service database-nya menggunakan Firebase. Aplikasi ini mempunyai 5 fungsional utama (tanpa menghitung Log In, Sign Up, dan Sign Out) yaitu:

  • Melihat istilah IT yang telah ditambahkan pengguna aplikasi
  • Menambah istilah IT
    1. Di halaman Home, tekan icon 3 titik yang berada di atas kanan layar
    2. Tekan "Add term"
    3. Tulis judul, kategori, dan deskripsi dari istilah yang ingin kamu tambahkan
    4. Setelah selesai, tekan tombol ceklis di kanan bawah
  • Menyunting/edit istilah IT yang telah kamu tambahkan
    1. Di halaman Home, tekan icon 3 titik yang berada di atas kanan layar
    2. Tekan "My Terms"
    3. pilih istilah yang ingin kamu edit lalu tekan
    4. Edit istilah kamu
    5. Setelah selesai, tekan tombol ceklis di kanan bawah
  • Menghapus istilah IT yang telah kamu tambahkan tambahkan
    1. Di halaman Home, tekan icon 3 titik yang berada di atas kanan layar
    2. Tekan "My Terms"
    3. pilih istilah yang ingin kamu hapus lalu tekan lama/hold
    4. tekan "Delete"
  • Mencari istilah IT yang telah ditambahkan oleh seluruh pengguna aplikasi
    1. Di halaman Home, ketik istilah yang ingin kamu cari di search bar lalu tekan icon search
    2. Lalu kamu dapat melihat hasil pencariannya

Source code : github.com/Akmalinnuha/Devterm

JetNote
